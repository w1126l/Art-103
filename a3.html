Assignment 3: System Readings

Art is not supposed to change the world, to change practical things, but to change perceptions. Art can change the way we see the world. Art can create an analogy. -- JR
Part 1 - Questions

1. What is a good working definition of what a System is and perhaps what it is not?
A system is a complex group working together.

2. For the 4 system definations you were to look up please give in your own words what they mean and give an example of them for each case.
Modularity means a system that can be rearranged. An easy example that I can think of is the Lovesac modular furniture. Decomposibility is the point at which elements cannot be broken down anymore. For example, H2O can be broken down into 2 Hydrogen atoms and 1 Oxygen atom. If we break it any further, the object would separate into protons, electrons, and neutrons, reaching functional obliteration. Emergence is the addition of new properties to a system through the multiplication of simple interactions in a wider group. An example would be hurricanes. The random motions of water molecules create a bigger thing. Chaos Theory is the theory that random systems with a wide-range of outcomes can still reach a level of pattern and balance. An example would be the double-rod pendulum.

3. What is the difference between ideas of Modularity and Tesselation? What are the properties that are in oppostion with each other?
Tesselations contain elements that are able to fit together through shared patterns and lines. However, they cannot be moved in the way that modular elements can. Modular elements are interchangeable.

4. What is the difference between designing something that has Complexity (aperiodic) vs being Uniform (periodic).
A design that is uniform, or periodic, is symmetrical. A complex, or aperiodic, design is inconsisten and asymmetrical, stepping closer into organized chaos.

5. What is the difference between a system diagram and a some kind of feature diagram like a like a exploded view drawing or a illustration like this lady bug. Please explain in complete sentances.
A system diagram shows how every element, whether visible or not, passes through a system. An exploded view shows the physical elements of a system in terms of different parts. For example, the ladybug is an exploded view drawing because it separates the ladybug into different parts and section, like wings and body. If it was a system diagram, it would show how things pass through the ladybig like wind, pollen, etc.
